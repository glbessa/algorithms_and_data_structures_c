# Algorithms and data structures in C

Hello, welcome to my repository. Here you will find some of the algorithms and data structures that I have implemented in C.

## How to use

To use one of this algorithms or data structures, you can just copy the .h file that you want to use and paste it in your project. Then, you can include it in your code with `#include "name_of_the_file.h"`.
